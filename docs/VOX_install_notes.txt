VOX integration steps

1) Put these HTML files into:
   C:\Users\tzvan\Desktop\VOX\docs\
   - docs.html   <- use docs_replacement.html / vox_docs_user_manual.html content
   - privacy.html

2) Update your main page footer links in index.html:
   - Documentation -> /docs
   - Privacy -> /privacy
   - Contacts -> /docs#uk-contact

3) Your current /docs route is FastAPI Swagger by default.
   Move Swagger to /api/docs and serve the custom HTML at /docs.

4) Add the POST /api/contact endpoint.
   Use fastapi_docs_contact_integration.py as the example.

5) For Gmail sending:
   - enable 2-Step Verification
   - create an App Password
   - set environment variables:
     GMAIL_USER
     GMAIL_APP_PASSWORD
     CONTACT_TO_EMAIL=avotiyaaa@gmail.com

6) Optional but recommended:
   - add rate limiting
   - add CAPTCHA / anti-spam
   - verify the final retention periods and subprocessors before publishing privacy.html
