Синхронний AI-переклад у реальному часі

VOX перетворює ваш смартфон на професійного синхронного перекладача. 15 мов, затримка ~1 секунда, учасникам потрібен лише браузер.

Як це працює

Від голосу до перекладу за 1 секунду

VOX використовує конвеєр з трьох AI-сервісів, які працюють паралельно. Затримка не залежить від кількості мов — всі переклади йдуть одночасно.

Deepgram Nova-2 ~300ms
Потокова транскрипція мовлення. Текст з'являється по мірі того, як ви говорите. Вбудований VAD, розділові знаки, 98%+ точність.
GPT-4o-mini ~700ms
Контекстний переклад з урахуванням попередніх фраз. Промпт враховує помилки розпізнавання і відновлює сенс. Паралельний переклад на N мов одночасно.
edge-tts ~300ms
Синтез мовлення на цільовій мові. Основний рушій безкоштовний (Microsoft). Автоматичний fallback на OpenAI TTS при збоях.
Режими

Solo та Room — два способи перекладу

🎧

Solo — персональний

Один мікрофон, один слухач. Оберіть мову джерела та перекладу, натисніть «Слухати» — і чуйте переклад у навушниках у реальному часі. Антиехо-система запобігає зацикленню.

🌐

Room — мультимовна кімната

Хост створює кімнату → QR-код → учасники сканують → кожен обирає свою мову → чує переклад. Хост керує: дає слово, вимикає, відключає. Один спікер у кожен момент.

✋

Протокол «Піднята рука»

Учасник натискає «Хочу сказати» → хост бачить запит → дозволяє або відхиляє. При дозволі мікрофон активується, голос перекладається для всіх.

Що робить VOX особливим

⚡

Швидкість

Затримка ~1 секунда. Проміжні результати з'являються миттєво, поки ви ще говорите.

🔗

Без додатку

Учасникам не потрібно нічого встановлювати. Сканують QR-код → відкривають сторінку в браузері → готово.

🎯

Контекстний переклад

AI враховує попередні 5 фраз розмови для точнішого перекладу. Автоматично виправляє помилки розпізнавання.

🎛️

Повний контроль

Хост керує всім: дає/забирає слово, вимикає звук, відключає учасників. Як дирижер оркестру.

🔊

Speaker Tracking

Всі бачать хто зараз говорить. Історія розмови з іменами спікерів і часовими мітками.

💳

Прозора тарифікація

Поповнення через Stripe. Попередження за 2 хвилини до вичерпання балансу. $3 бонус за верифікацію email.

API

WebSocket та REST API

VOX працює через WebSocket для аудіо-потоку та REST API для управління. Нижче — основні ендпоінти.

Метод Ендпоінт Опис
WS /ws/solo Solo-режим: аудіо → транскрипція → переклад → TTS
WS /ws/room/{id}/host Хост кімнати: аудіо + управління учасниками
WS /ws/room/{id}/guest/{gid} Гість кімнати: отримує переклад, може говорити при дозволі
POST /room/create Створити кімнату → room_id + QR-код
POST /room/{id}/join Приєднатися до кімнати
POST /api/register Реєстрація нового користувача
POST /api/login Авторизація → session token
GET /api/balance Поточний баланс користувача
POST /api/create-checkout Створити Stripe Checkout Session
GET /api/reviews/public Публічні відгуки

Побудовано на

🐍

FastAPI + Uvicorn

Асинхронний Python-сервер. WebSocket + REST API. Розгортання на Railway.

🎤

Deepgram Nova-2

Streaming STT з проміжними результатами. Точність 98%+, вбудований VAD, ~300мс затримка.

🧠

GPT-4o-mini

Контекстний переклад з rolling history. ASR-aware промпт, що враховує помилки розпізнавання.

🔊

edge-tts + OpenAI TTS

Безкоштовний Microsoft TTS як основний рушій. OpenAI TTS tts-1 як автоматичний fallback.

💳

Stripe

Checkout Sessions для поповнення балансу. Webhooks для автоматичного зарахування коштів.

🌐

Web Audio API

AudioWorklet для захоплення мікрофона. Web Audio API для відтворення TTS. Без фреймворків.
